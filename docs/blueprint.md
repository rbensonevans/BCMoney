# **App Name**: BCMoney

## Core Features:

- User Authentication: Secure user login and registration using credentials.
- Balance Display: Display the user's current balance upon successful login.
- Transaction History: Display a list of the user's recent transactions with dates and amounts.
- Deposit: Display a unique generated ethereum address so crypto can be sent to that address
- Withdraw: Allows the user to transfer funds from the account to an ethereum address.
- Send: Allow the user to send funds from the account to another users
- Profile: Allow the user to update his profile with a photo, first, middle and last name, address, phone number and email address.
- Unique Name: Allow the user to specify a unique name which is linked to his account number and an ethereum public address. This should also be stored in a database.
- Recipients: Create 3 lists under Recipients. First a Family List, second a friends list and third a General list. This data will also be stored in a database. An add and delete button should be added to manage list elements.

## Style Guidelines:

- Primary color: Dark blue (#1A237E) to convey trust and security.
- Background color: Light gray (#EEEEEE) for a clean, modern look.
- Accent color: Teal (#008080) for interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif font.
- Use minimalist, outline-style icons for transactions and menu items.
- A clean and intuitive layout with a focus on usability.
- Subtle transitions for page navigation and loading states.
